https://hypermedia.systems/

Building hypermedia apps with HTMX, HXML, Go and Templ.

## Run the web example
```shell
make up
```

## Dev
### Install Templ and WGO command line tools.
```shell
make dev.setupd
```
### Run wgo with templ file watching to auto-reload changes
```shell
make dev
```
